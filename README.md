# CAAFEMOD 
Minecraft quality of life improvement mod by Caafe :) 

## Features
Custom: 
  - blocks
  - items
  - tools
  - foods
  - tags
  - recipes
  - drop tables
  - slabs and stairs
  - walls, fences, fence doors
  - doors, trapdoors
  - buttons, pressure plates
  - creative mode tab

Which led me to explore: 
  - 3D-2D models
  - textures
  - pixel art
  - data generation
